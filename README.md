# UNILIB
Este repositorio es un prototipo para el sistema de gestión bibliotecario UNILIB, proyecto semestral para el ramo de Ingenieria de Software, sección 008D. El grupo de proyecto está conformado por:
- José Calderón
- Sebastian Apablaza
- Miguel Bravo

El prototipo fue creado con la herramienta Figma Make, el cual esta disponible en [https://www.figma.com/design/CPSFM9AIiX2wCCIqLaRfgQ/UNILIB](https://www.figma.com/design/CPSFM9AIiX2wCCIqLaRfgQ/UNILIB).

**En la carpeta `docs` se encuentran los documentos de entregas anteriores y la actual (final).**

## Funciones principales del proyecto

- Panel de inicio de sesión con cuentas de prueba
- Catálogo de libros
- Dashboard / Pantalla de inicio
- Gestión de libros
- Gestión de usuarios
- Gestión de prestamos / reservas / adquisiciones

## Correr el código
Para correr el código, debe tener instalado Node.js, dentro del directorio debe correr los comandos `npm i` para instalar las dependencias y  `npm run dev` para iniciar el servidor de desarrollo.
